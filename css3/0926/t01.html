Display 속성

각각의 요소들을 어떻게 보여줄지 결정한다

속성 : visiblity(기본값:보이기) / hidden(감추기) / inherit(부모요소의 값을 상속받아서 사용 위에서 지정해놓은 속성(글자크기 간격등)을 가져온다)

Display 속성 none : 화면에서 완전히 사라지게 됩니다.
